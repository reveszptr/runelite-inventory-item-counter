# 🧮 Inventory Item Counter (RuneLite Plugin)

A custom RuneLite plugin written in Java that counts a selected item in the player's inventory and displays the result in a clean overlay.

---

## 🚀 Features

- Count a selected item using **Item ID**
- Real-time updates (every game tick)
- Simple and clean **overlay display**
- Toggle overlay visibility via plugin config
- Lightweight and beginner-friendly plugin structure
